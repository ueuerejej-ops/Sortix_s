# Sortix_s
# Rust File Sorter

Fast file organizer with GUI built using egui.

## 🚀 Features

* Sort files by type (images, videos, music, documents)
* Code mode (Rust, Python, C, JS, etc.)
* Undo sorting
* Multi-threaded file processing



(вставишь скрин своего приложения)

## ⚙️ How it works

1. Choose a folder
2. Select file types
3. Click "Sort"
4. Files are automatically organized into folders

## 🛠 Tech Stack

* Rust
* egui (GUI)
* std::fs

## 📦 Run

```bash
cargo run
```
